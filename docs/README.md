#  Detecção de Formigueiros com U-Net

Projeto de segmentação semântica para identificar formigueiros em imagens aéreas.

## Objetivo
Detectar automaticamente formigueiros em imagens RGB de drone.

## Abordagem
- Modelo: U-Net
- Tipo: Segmentação binária (fundo vs formigueiro)
- Técnica principal: Patch Training
